# RandR Backend (Typescript conversion)

This is using the same project as last sprint. but now I have converted the entire project to use typescript. This was a much heavier lift than I anticipated, but it was a great learning experience.

## Instructions for Build and Use

[Software Demo](https://youtu.be/TH6vo2gWJ0s)

Steps to build and/or run the software:

1. run npm i to get all dependencies
2. run tsc to compile the typescript
3. run npm start to start the project

Instructions for using the software:

1. use the testing.rest file to send sample requests for the user routes
2. open localhost:3000/exercise to see the exercise routes.
3. PUT, POST, and DELETE routes have also been setup on the exercise route, you can use an ID from the get route to test those routes
